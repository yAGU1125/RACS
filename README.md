# NFC入退室管理システム (NFC Room Monitor System)

スマートフォンやPCからアクセス可能なWeb UIと、Discord Bot連携機能を備えたサークル・研究室向けの入退室管理システムです。学生証や交通系ICカードをタッチするだけで、入退室の記録とDiscordへのリアルタイム通知が行われます。

## ✨ 主な機能 (Features)

* **NFCスマート登録:** 学生証（FeliCa）の学籍番号読み取りに加え、Suica等の交通系ICカードやスマートフォンの物理ID読み取りにも対応。
* **Webダッシュボード:** モバイル端末からも操作しやすいUIで、現在の入室状況をリアルタイムで確認可能。
* **Discord Bot連携:**
  * 入退室時の自動通知機能（Webhook）
  * チャンネル内での自然言語による在室確認コマンド
* **自動ログ管理:** 日別の入退室履歴をCSVとして自動保存し、7日以上経過した古いログを自動的に削除します。

## 🤖 Discordコマンドの使い方 (Usage)

Discordのチャンネル内で以下のメッセージを送信すると、Botが現在の在室状況を返信します。

* `/在室人数` : 現在の部屋全体の入室者数と一覧を表示します。
* `[特定のキーワード]の在室人数` : 専攻、チーム、学年などで絞り込み検索が可能です。
  * *(例)* `情報工学科の在室人数`
  * *(例)* `B4の在室人数`
  * *(例)* `イベントチームの在室人数`
* `[役職名]の在室確認` : 特定の役職を持つメンバーが在室しているか確認します。
  * *(例)* `広報リーダーの在室確認`

## 🛠 動作環境 (Requirements)

* Python 3.x
* NFCカードリーダー (PaSoRi RC-S380 等)
* 必須ライブラリ: `flask`, `discord.py`, `nfcpy`, `requests`, `pandas`
